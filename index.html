Explora el mundo del desarrollo web

Bienvenido a la Tienda GitHub, un proyecto académico orientado a la práctica de conceptos fundamentales del desarrollo de aplicaciones web. Aquí se integran conocimientos básicos de HTML y estructura semántica.

Este sitio sirve como punto de partida para el aprendizaje de tecnologías web modernas y la correcta organización de contenidos.

Sobre el proyecto

Este proyecto forma parte de la asignatura Aplicaciones Web I y tiene como objetivo fortalecer el uso de etiquetas HTML, navegación básica y buenas prácticas en el desarrollo web.
